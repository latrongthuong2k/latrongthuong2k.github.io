こんにちは

私はトゥオンです。私はゲームプログラマーであり、ゲームデザイナーです。ゲームを構築し、創造することに熱中し、私が作成したゲームをプレイしたときにプレイヤーに素晴らしい感動を与えることを願っています。そのモチベーションによって、私は毎日新しい技術を研究し、学び、試して、将来は優秀な人々と一緒に働けることを望んでいます。

私のプロジェクト

Gravity Jump (2021)

Game Design

Code

これは、ゲームハッカソンのクラスに参加した1週間で自分で作成したゲームです。ゲームはとてもシンプルですが、プレイヤーが最終的な目的地に最小の時間で赤いバーに触れないように努力するときには少し魅力的です。キャラクターの速度がとても速いので、プレイヤーは失敗しないように注意しなければなりません。
